Thresholds

Classification: peak activity when a peak exceeds the peak threshold; still when all axis SDs stay below the stillness threshold and ODBA stays below the active threshold; otherwise active. Peak acceleration is the highest raw vector magnitude above the 1 g gravity baseline.

CSV export

Exports contain the current browser session while the measurement buffer has space. ODBA and VeDBA are relative activity proxies; calibrate them for species, tag position, and behaviour before biological quantitative interpretation.
